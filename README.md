# Supplementary Data for: Genomic profiling of biofilm‑associated genes in clinical *Staphylococcus aureus* from Nigeria

This repository contains all supplementary data, analysis scripts, and results for the manuscript:

**Genomic profiling of biofilm‑associated genes in clinical *Staphylococcus aureus* from Nigeria: association with multidrug resistance and clonal lineages**

Authors: Brown Beckley, Vincent Amarh, Innocent Afeke, Adesola Olalekan

## Contents

- **Supplementary Tables (CSV)**
  - `Supplementary_Table_S1_metadata.csv` – Clinical metadata (age, sex, sample source, hospital, diagnosis) for 107 isolates.
  - `Supplementary_Table_S2_Dunn_test.csv` – Pairwise Dunn's post‑hoc comparisons for biofilm score by MLST.
  - `Supplementary_Table_S3_correlations.csv` – Spearman correlations between biofilm score and AMR functional classes.
  - `Supplementary_Table_S4_Youden.csv` & `Supplementary_Table_S4_confusion_matrix.csv` – Youden cut‑off and confusion matrix.
  - `Supplementary_Table_S5_Firth.csv` – Full output of Firth’s penalised logistic regression.

- **Supplementary Figure**
  - `Supplementary_Figure_S5.pdf` – Heatmap of biofilm gene presence/absence for major STs (≥5 isolates).

- **Raw StaphScope output files**
  - `amr_genes.csv`, `virulence_genes.csv`, `bacmet_genes.csv`, `sample_overview.csv`, `biofilm_analysis_dataset.csv`

- **All R scripts used for statistical analysis and figure generation**
  - `analysis_*.R` and `generate_supp_table_*.R`

- **README.txt** – Original plain‑text description (same as above)
